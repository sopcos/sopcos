# Sopcos Specifications (SIPs)

This repository contains the official **Sopcos Improvement Proposals (SIPs)** and technical specifications.
It serves as the **"Single Source of Truth"** for the Sopcos Protocol, Governance, and Standards.

## Status
- **SIP-001:** Sopcos Policy Language (SPL) - *Draft*
- **SIP-002:** Registry Specification - *Planned*
- **SIP-003:** Policy Lifecycle - *Planned*

## Governance
Changes to the protocol must be submitted as a SIP via Pull Request.
