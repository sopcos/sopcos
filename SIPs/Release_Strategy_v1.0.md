## 1. Purpose of This Release

SOPCOS v1.0 is **not** a feature-complete platform release.  
It is a **constitutional release**.

The objective of v1.0 is to:

- Fix **responsibility boundaries**

## 2. Release Philosophy

### 2.1 Governance-First Principle

Most protocols ship code first and governance later.  
SOPCOS deliberately inverts this order.

**v1.0 ships:**
- Who decides
- How conflicts are resolved
- Who is liable
- What evidence is immutable

**v1.0 does NOT ship:**
- Growth incentives
- Permissionless chaos
- Autonomous authority
